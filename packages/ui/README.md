# @ishopine/ui

Polaris-inspired React primitives for iShopine dashboards.

Requires `@ishopine/design-system/tokens.css` in the app.

## Components

- `Button` — primary / secondary / destructive / plain / dark
- `Card`, `CardHeader`, `CardTitle`, `CardDescription`
- `Input`, `Badge`
- `PageHeader`, `EmptyState`, `LoadingState`, `ErrorState`, `SuccessBanner`
- `AdminShell` — dark top bar + light sidebar + content canvas

```tsx
import { AdminShell, PageHeader, Card, Button } from "@ishopine/ui";
```
